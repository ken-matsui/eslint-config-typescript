# eslint-config-typescript

## Installation

1. Run the install command
   ```bash
   $ npm install -D @ken-matsui/eslint-config-typescript
   ```
2. You also need to install peer dependencies
   ```bash
   $ npm install -D eslint
   ```

## Usage

`.eslintrc.yml`

```yaml
extends:
  - "@ken-matsui/eslint-config-typescript"
```
